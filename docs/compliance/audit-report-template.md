## 五、风险矩阵

### 风险评估标准

| 严重程度 | 定义 | 示例 |
|---------|------|------|
| **Critical** | 直接导致数据泄露/系统被控/合规违规 | 未加密的 PII 存储、未认证的 API 端点 |
| **High** | 可能导致重大安全事故，需特定条件 | 缺少 MFA、审计日志可被删除 |
| **Medium** | 增加安全风险但不直接导致事故 | 会话超时过长、错误信息泄露 |
| **Low** | 最佳实践偏离，风险较小 | 文档不完整、非关键配置未加固 |
